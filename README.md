# Smart Hiring Platform

A full-stack recruitment platform built with React, Spring Boot, and MySQL.

## Features
- User Authentication
- Candidate Profile Management
- Recruiter Dashboard
- Job Posting
- Job Application System
- Resume Upload
- ATS Resume Matching
- Interview Scheduling
- Notifications
- Admin Dashboard

## Tech Stack
Frontend: React
Backend: Spring Boot
Database: MySQL
Security: Spring Security + JWT
ORM: Spring Data JPA + Hibernate

## Project Goal
To build an industry-level hiring platform for real-world recruitment workflow.
